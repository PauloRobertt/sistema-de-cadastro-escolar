# 📚 Sistema de Cadastro Escolar

É um sistema de gerenciamento de usuários e alunos, que permite o cadastro, autenticação e controle de acesso de usuários, além do cadastro de alunos com possibilidade de envio de imagem de perfil.

O sistema possui funcionalidades completas de CRUD tanto para os usuários quanto para os alunos, com upload e armazenamento de imagens para perfis de estudantes. Ideal para instituições de ensino que desejam manter um controle simples e eficiente de seus alunos e administradores.
